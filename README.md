College Portal Clone

A responsive front-end recreation of a professional college portal, developed during my first internship to improve user interface design and navigation.

Key Features:

Responsive Design: Fully optimized for mobile, tablet, and desktop views using CSS Media Queries.

Interactive UI: Clean navigation menus and structured layouts for academic data.

Cross-Browser Compatibility: Tested to ensure a consistent experience across different web browsers.

Tech Stack: HTML5, CSS3, JavaScript Basics, Bootstrap.
